Os parâmetros obtidos servem de apoio para avaliar as vizinhanças que tiveram melhor desempenho na execução. O ranking F(factor), foi obtido com a somatória de todos os fatores(improvement / time) das vizinhanças de cada instância. Da mesma forma, o ranking I(improvement), foi obtido com a somatória das melhorias das vizinhanças de cada instância. O campo (% factor) é a normalização do valor do fator. Para obtê-lo, divimos cada um dos fatores pelo maior fator.
